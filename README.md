# screen-saver-cli

[![license](https://img.shields.io/github/license/vutran/screen-saver-cli.svg?maxAge=2592000&style=flat-square)](LICENSE)

> Starts the screensaver via the command-line.

## Install

```bash
$ npm install -g screen-saver-cli
```

## Usage

```bash
$ screen-saver
```
