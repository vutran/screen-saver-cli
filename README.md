# screen-saver-cli

[![MIT licensed](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

> Starts the screensaver via the command-line.

## Install

```bash
$ npm install -g screen-saver-cli
```

## Usage

```bash
$ screen-saver
```
